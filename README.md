## Getting Started
Follow these steps to get a local copy of the project up and running:

### Prerequisites
- Node.js >= 14.x
- npm >= 6.x

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/Influwealth/argus-prime.git
   ```
2. Navigate to the project directory:
   ```bash
   cd argus-prime
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
